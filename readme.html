What it does

Your store currently makes use of Zen Cart's "Linked Products" feature. This plugin provides an admin-level report, accessible via Reports->Linked Products, that identifies, for each linked product:

  1. The name of the product's Master Category, with a link to that category. If the master_category_id for the product is invalid, this column shows ==> Invalid categories ID (xx), where xx is the invalid category ID.
  2. The name of the product, with a link to that product's definition.
  3. A list of Linked Categories, each with a link to that category.

The table is sorted by the Master Category name.

Installation

There are no core-file overwrites in this plugin; you should always backup your cart’s database and files prior to making any changes. Unzip the plugin's package file, rename the YOUR_ADMIN folder to match your custom admin folder's name and copy the plugin's files to your admin folder:

  1. /YOUR_ADMIN/stats_products_linked.php
  2. /YOUR_ADMIN/includes/extra_datafiles/stats_products_linked_filenames.php
  3. /YOUR_ADMIN/includes/functions/extra_functions/init_stats_products_linked.php
  4. /YOUR_ADMIN/includes/languages/english/stats_products_linked.php
  5. /YOUR_ADMIN/includes/languages/english/extra_definitions/stats_products_linked_extra_definitions.php

Uninstall

First, delete the files that were copied to your zen-cart installation's admin directories. Restore your backup up of then use your admin's Tools->Run SQL Patches to run the file stats_products_linked_uninstall.sql.
